石原慎太郎のファイルのみ圧縮されており，いくつか処理にあたって問題があった．
 - url が quote されたままだったので，urllib.unquote() を使って処理した．
 - quote された url には href="/wiki/名前" となっていたので，/wiki/ を除いた．
 - この処理のために，
     - kde の日本語の package を install する必要があった
     - emacs の default 設定に
       (set-buffer-file-coding-system 'utf-8)
       (set-default-coding-systems 'utf-8)
       (set-terminal-coding-system 'utf-8)
       を加えた．(そのため EUC-JP ではなくなる．)
